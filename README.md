## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sefakrb/g-tower.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build Setup 
   * Run on development mode
     ```sh
     npm run dev
     ```
   * Build for production and launch server
      ```sh
      npm run build
      npm run start
      ```
